# BIOSTAT721_Project2

This repository contains the R project for BIOSTAT 721 Project 2.
The project processes daily air pollution data (CO, PM2.5, O3) from 2018–2020 and produces two required plots.

Contents:

  BIOSTAT721_Project_2_Weizhao_Liu.Rproj
  
  Project2.Rmd – main R Markdown report (knits to Word)
  
  data/ – raw air quality datasets (AQ_2018.csv, AQ_2019.csv, AQ_2020.csv)
  
  README.md

Description:

  The project includes:
  
    A function that cleans the air quality data by renaming variables, formatting dates, removing duplicates, averaging multiple measurements per day, and correcting negative values.
    
  Two plots:
  
    Monthly CO and O3 averages (with 95% CI).
    
    Monthly PM2.5 concentrations by year.
